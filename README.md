# titanic
Resolução em Python do desafio kaggle com dataset titanic


Estudo comparando duas abordagens, uma com apenas uma rede neural para prever se um passageiro irá morrer ou não e uma outra abordagem utilizando duas redes, uma para o sexo feminino e outra para o sexo masculino. Essa ideia surgiu por conta da grande diferença entre as probabilidades de sobrevivencia baseadas no sexo do passageiro.

Ainda está sob desenvolvimento.
